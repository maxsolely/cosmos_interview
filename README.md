## How to run

This is a next.js app that runs an apollo server and consumes a graphql api. Both client side and server side is included in this repo.

To run this app, install the dependencies and then run

```bash
npm run dev:all
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
